# hospital-management-db
PostgreSQL Hospital Management Database Project
## Overview
This project is a Hospital Management Database System built using PostgreSQL. It demostrates databse design, SQL queries, and real-world data management conceptts used in healthcare systems.
----
## Features
-Patient management system
-Doctor management system
-Appointment scheduling 
-Database relationship using Foreign Keys
-SQL operations (CRUD: Create, Read, Update, Delete)
-Data backup and recovery
---
## Database Tables
### 1. Patients
Store patient information such as name, gender, contact and address.
### 2. Doctors
Store doctor details including specialization and contact information.
### 3. Appointments
Connects patients and doctors using foreign keys and manages appointment scheduling.
